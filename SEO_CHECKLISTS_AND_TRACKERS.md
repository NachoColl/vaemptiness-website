## 1. Pre-Launch SEO Checklist

### Technical Foundation
- [ ] Google Search Console verified
- [ ] Google Analytics 4 installed and configured
- [ ] XML sitemap generated
- [ ] Sitemap submitted to Google Search Console
- [ ] Robots.txt file created and accessible
- [ ] HTTPS enabled and enforced
- [ ] Canonical tags on all pages
- [ ] 404 error page exists and is helpful
- [ ] Page load speed <3 seconds (desktop)
- [ ] Page load speed <5 seconds (mobile)
- [ ] Mobile-friendly test passed (all pages)

### On-Page SEO
- [ ] All pages have unique title tags (<60 chars)
- [ ] All pages have unique meta descriptions (<155 chars)
- [ ] All pages have one H1 tag
- [ ] Heading hierarchy is logical (H1 > H2 > H3)
- [ ] All images have descriptive alt tags
- [ ] All images have optimized file names
- [ ] Internal linking structure implemented
- [ ] No broken links (check with tool)
- [ ] URL structure is clean and descriptive

### Schema Markup
- [ ] LocalBusiness schema on all pages
- [ ] MedicalBusiness schema on homepage
- [ ] Person schema on about page
- [ ] PsychologicalTreatment schema on program pages
- [ ] FAQPage schema on FAQ page
- [ ] BlogPosting schema on blog posts
- [ ] BreadcrumbList schema on applicable pages
- [ ] WebSite schema on homepage
- [ ] All schema tested with Google Rich Results Test

### Content Quality
- [ ] Homepage has 500+ words
- [ ] Each program page has 800+ words
- [ ] About page has 600+ words
- [ ] Content is original (not copied)
- [ ] No spelling or grammar errors
- [ ] Content includes target keywords naturally
- [ ] Content provides value to users
- [ ] Content demonstrates E-E-A-T (expertise)

### Local SEO
- [ ] Google Business Profile created
- [ ] GBP fully optimized (all fields complete)
- [ ] GBP photos uploaded (minimum 10)
- [ ] GBP verified
- [ ] Local citations submitted (minimum 10)
- [ ] NAP (Name, Address, Phone) consistent everywhere
- [ ] Local keywords in content
- [ ] Address in footer (if applicable)

### Open Graph & Social
- [ ] Open Graph tags on all pages
- [ ] Twitter Card tags on all pages
- [ ] Social media profiles created
- [ ] Social media profiles linked from site
- [ ] Site linked from social media profiles
- [ ] Default social share images created (1200x630px)

## 11. Emergency Troubleshooting Checklist

### Sudden Traffic Drop

If organic traffic drops >20% week-over-week:

- [ ] Check Google Search Console for manual actions
- [ ] Review recent algorithm updates (Google, MozCast)
- [ ] Check site is accessible (not down or broken)
- [ ] Verify sitemap is still submitted
- [ ] Check for broken links or crawl errors
- [ ] Review recent content changes (did you delete pages?)
- [ ] Check competitors (industry-wide drop?)
- [ ] Review Analytics for data accuracy issues

### Rankings Drop for Important Keywords

- [ ] Check if page is still indexed (site:vaemptiness.es)
- [ ] Review on-page SEO (title, meta, content)
- [ ] Check for duplicate content issues
- [ ] Analyze SERP (what changed? new competitors?)
- [ ] Review backlink profile (lost important links?)
- [ ] Check mobile-friendliness
- [ ] Review page speed (did it slow down?)

### Google Business Profile Issues

- [ ] Verify profile is still verified
- [ ] Check for policy violations
- [ ] Ensure NAP is consistent
- [ ] Review categories (still appropriate?)
- [ ] Check hours are correct
- [ ] Verify photos are still visible
- [ ] Check for duplicate listings
