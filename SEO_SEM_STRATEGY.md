## Executive Summary

This document outlines a comprehensive SEO/SEM strategy for vaemptiness.es to compete effectively in Barcelona's mental health and mindfulness therapy market. The strategy focuses on local SEO optimization, content targeting, schema markup implementation, and paid advertising campaigns to attract clients seeking mental training and mindfulness-based therapeutic services.

**Key Differentiators:**
- Buddhist philosophy integration
- Mental training ("entrenamiento mental") rather than traditional therapy
- Mindfulness and cognitive de-cluttering focus
- Programs for adults, teams, teens, and kids

### 3. Technical SEO Requirements

#### Site-Wide Technical Elements

**1. XML Sitemap**
- Generate sitemap.xml with all pages
- Submit to Google Search Console
- Update frequency: weekly for blog, monthly for static pages

**2. Robots.txt**
```
User-agent: *
Allow: /
Disallow: /page-reviews/
Disallow: /scripts/

Sitemap: https://vaemptiness.es/sitemap.xml
```

**3. Structured Data (Schema Markup)**
- LocalBusiness schema on all pages
- MedicalBusiness schema on homepage
- Person schema for team members
- PsychologicalTreatment schema for program pages
- FAQPage schema on FAQ page
- BlogPosting schema for blog articles
- BreadcrumbList schema for navigation

(See separate schema files)

**4. Open Graph Tags (Social Sharing)**

All pages should include:
```html
<meta property="og:title" content="[Page Title]">
<meta property="og:description" content="[Meta Description]">
<meta property="og:image" content="https://vaemptiness.es/assets/images/og-image.jpg">
<meta property="og:url" content="https://vaemptiness.es/[page-url]">
<meta property="og:type" content="website">
<meta property="og:locale" content="es_ES">
```

**5. Twitter Cards**
```html
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="[Page Title]">
<meta name="twitter:description" content="[Meta Description]">
<meta name="twitter:image" content="https://vaemptiness.es/assets/images/twitter-card.jpg">
```

**6. Canonical Tags**
All pages should have canonical URL:
```html
<link rel="canonical" href="https://vaemptiness.es/[page-url]/">
```

**7. Hreflang (Future Multi-language)**
If you add English/Catalan in future:
```html
<link rel="alternate" hreflang="es" href="https://vaemptiness.es/">
<link rel="alternate" hreflang="en" href="https://vaemptiness.es/en/">
<link rel="alternate" hreflang="ca" href="https://vaemptiness.es/ca/">
```
